## Custom CSS

For edit and add new color on your theme:

- key: `value` for use uniq color for light / dark scheme
- key: `{light: {value}, dark: {value}}` for use uniq color for light / dark scheme
-

```javascript
export default {
	//...
	themes: {
		default: {
			primary: { light: 'blue', dark: 'red' }
		}
	}
	//...
};
```
